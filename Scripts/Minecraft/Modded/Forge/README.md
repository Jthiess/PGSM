# Minecraft Java — Forge Install Script

**Script**: `install-mcforg.sh`
**Server type**: `forge`
**Game code**: `MCJAV`

---

## What It Does

1. Updates and upgrades the container (`apt update && apt upgrade -y`)
2. Downloads Java 8, 16, 17, 21, and 25 (Eclipse Temurin JREs) to `/opt/java/`
3. Extracts and renames each Java version
4. Downloads the Forge installer JAR (from `forge_url` or `serverfilelink`) to `/PGSM/forge-installer.jar`
5. Runs the Forge installer: `/opt/java/java21/bin/java -jar forge-installer.jar --installServer`
6. Resolves the startup command from the generated `unix_args.txt` at `libraries/net/minecraftforge/forge/*/unix_args.txt`
7. Creates the `PGSM` system user, sets ownership, and symlinks the selected Java to `/usr/local/bin/java`
8. Writes `eula=true` to `/PGSM/eula.txt`
9. Installs tmux
10. Writes and enables the `PGSM` systemd service (Type=forking, starts the Forge server in a tmux session named `PGSM`)

---

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `serverfilelink` | **Yes** | URL to the Forge installer JAR. Used as fallback if `forge_url` is not set. Script exits with an error if omitted. |
| `forge_url` | No | Direct URL to the Forge installer JAR. Takes priority over `serverfilelink`. |
| `java_version` | No | Java major version for the startup command: `8`, `16`, `17`, `21`, or `25`. Defaults to `21`. |
| `startup_command` | No | Override the full startup command (skips `unix_args.txt` resolution). |
| `type` | No | Server type string (always `forge` for this script) |

---

## Default Startup Command

```
/opt/java/java<version>/bin/java -Xms512M -Xmx2G -XX:+UseG1GC @/PGSM/libraries/net/minecraftforge/forge/<version>/unix_args.txt nogui
```

The `unix_args.txt` file is generated by the Forge installer and contains the full classpath and JVM arguments for that Forge version.

---

## Notes

- The Forge installer always runs with Java 21 regardless of the `java_version` argument. The `java_version` argument only affects the startup command and the `/usr/local/bin/java` symlink.
- If `unix_args.txt` cannot be found after installation, the script exits with an error.
- `server.properties` is written separately by PGSM via SFTP after the script completes.
- Forge versions are resolved via the Forge promotions API (`https://files.minecraftforge.net/net/minecraftforge/forge/promotions_slim.json`) by `MinecraftService.get_forge_versions()`.
