# Minecraft Vanilla Install Scripts

This directory contains install scripts for vanilla (unmodded) Minecraft servers.

| Subdirectory | Edition | Script |
|-------------|---------|--------|
| `Java/` | Minecraft Java Edition | `install-mcjava.sh` |
| `Bedrock/` | Minecraft Bedrock Edition | `install-mcbedr.sh` |

See each subdirectory's README for script arguments and details.
