# Minecraft Notes
Java versions should be used as follows:
| Minecraft Version | Java Version |
| ----------------- | ------------ |
| ≤ 1.16.5          | Java 8       |
| 1.17              | Java 16      |
| ≥ 1.18            | Java 17      |

I will be using temurin binaries for the JRE, stored in /opt

## Install Process
1. update and upgrade
2. Download and install java versions
3. Extract java versions
4. Download server files
5. Create user to run server

## User Options
1. Version
2. Type (Paper, Forge, Fabric)
3. Render Distance
4. MOTD
5. Spawn Protection
6. difficulty
7. hardcore
